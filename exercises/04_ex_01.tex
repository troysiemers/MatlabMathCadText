{The double logistic curve is defined by 
\[
y=\textrm{sign}(x-5)\cdot\left(1-\exp\left[-\left(\frac{x-5}{50}\right)^2\right]\right)
\]
Use the command \cour{linspace} to get $x$ values from $-100$ to $100$ and plot $y$ for these $x$ values.  Add appropriate labels (something simple) to the axes and a title.}
